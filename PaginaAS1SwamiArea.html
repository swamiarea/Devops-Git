Microsoft Sharepoint

O SharePoint iniciou sua jornada em 2001 e desde então tem sofrido inúmeras alterações resultando num salto gigante de funcionalidades e melhorias de interface. O foco principal sempre foi favorecer a colaboração de equipes, permitindo uma forma fácil de compartilhar, organizar e acessar informações, ou seja, um verdadeiro portal de informações corporativas. Uma das grandes propostas do SharePoint é também facilitar a interação de usuários e Web Designers que possuem pouco conhecimento de TI com a ferramenta, possibilitando a criação de elementos de site com grande facilidade.

Microsoft Power BI

A análise de dados é algo primordial para qualquer tipo de tomada de decisão, seja ela profissional ou pessoal. Para isso, torna-se necessário termos o auxílio de uma ferramenta que nos ajude a realizar tal tarefa. O Power BI foi projeto em 2010, porém, só se tornou formalmente Power BI em 2013. O Microsoft Power BI é uma poderosa plataforma de Business Intelligence que une diversas ferramentas para análise de dados. Seu principal objetivo é a elaboração de relatórios e painéis (dashboards). O Power BI também te proporciona uma visão 360º com suas métricas mais importantes em só lugar, podendo ser atualizadas em tempo real e disponíveis em todos os seus dispositivos.

Microsoft Powershell

O PowerShell é uma solução de automação de tarefas multiplataforma que consiste em um shell de linha de comando, em uma linguagem de script e uma estrutura de gerenciamento de configuração. O PowerShell pode ser executado no Windows, Linux e macOS. O PowerShell é um shell de comando moderno que inclui os melhores recursos de outros shells populares. Ao contrário da maioria dos shells que só aceita e retorna texto, o PowerShell aceita e retorna objetos .NET. Como uma linguagem de script, o PowerShell é normalmente usado para automatizar o gerenciamento de sistemas. Ele também é usado para compilar, testar e implantar soluções, geralmente em ambientes de CI/CD. O PowerShell é criado no CLR (Common Language Runtime) do .NET. Todas as entradas e saídas são objetos .NET. Não é necessário analisar a saída de texto para extrair informações da saída.
